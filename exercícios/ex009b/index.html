Trabalhando com listas

Listas ordenadas

Para fazer uma lista ordenada, é necessário usar a tag <ol> e dentro dela a tag <li> para cada item.

  1. Acordar
  2. Tomar café
  3. Fazer PTI

Dentro dessa lista, você pode adicionar a funcionalidade type que te dá variações de numerações, exemplo: <ol type:"I">. Além disso, caso você deseje começar por outra numeração basta aadicionar o start, exemplo: <ol type:"I" start="2">

Listas não ordenadas

Para fazer uma lista ordenada, é necessário usar a tag <ul> e dentro dela a tag <li> para cada item.

  • Pão
  • Garrafa
  • Faca
  • Manteiga

Dentro dessa lista, você pode adicionar a funcionalidade type que te dá variações simbolos, exemplo: <ol type:"square">.

Lista de Definições

Para fazer uma lista de definição, é necessário usar a tag <dt> para citar o elemento e a tag <dd> para adicionar a descrição, elas não precisam estar uma dentro da outra, mas precisam estar nessa ordem.

HTML
Linguagem de marcação para a criação de um conteúdo de um site.
CSS
Linguagem de marcação para a criação de um design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.
